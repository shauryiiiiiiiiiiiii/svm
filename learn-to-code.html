SVM CODER HUB - LEARN HOW TO CODE HTML


HTML stands for HyperText Markup Language. It is used to create webpages containing both text and graphics
It was previously known as WYSIWYG - what you see is what you get.

Topic 1: ELEMENTS
An element consists of ON Tag, the contant and OFF Tag.
Elements can be catogorised as:
1. Empty Element
2. Container Element
An EMPTY element only contains ON Tag and do not have an OFF Tag.

Example: < br >


A CONTAINER element consists of both ON and OFF Tags.

Example: < b >BOLD TEXT< /b >


Topic 2: HEADINGS
Headings are used to give importance to a sentence. From h1 to h6.
H1 is used when the heading is so much important
H6 is used when the heading is not important at all.

Example:< h1 >IMPORTANT< /h1 >


Example: < h6 >USELESS< /h6 >


Topic 3: INSERTING IMAGES IN A WEBPAGE
To insert a image in webpage, we need to use < img src="image link here."> Tag.

Example: < img src="https://teddy.png">


Topic 4: HYPERLINKS
HYPERLINKS are used to redirect a user from one page to another by clicking on a normal text.

Example: < a href="google.com">Click me to go to google< /a >
